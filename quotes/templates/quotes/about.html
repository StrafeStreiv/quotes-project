️ О проекте Random Quotes

О чём этот проект?

Это веб-приложение для любителей цитат из фильмов и книг. Здесь вы можете:

  • Наслаждаться случайными цитатами
  • Добавлять свои любимые цитаты
  • Оценивать понравившиеся цитаты
  • Смотреть рейтинг популярных цитат

Возможности

Умный случайный выбор цитат (с учетом веса)
Добавление цитат через удобную форму
Защита от дубликатов
Не более 3 цитат на один источник
Система лайков/дизлайков
Топ-10 популярных цитат
Статистика и аналитика

Технологии

Backend: Django 4.2, Python 3.9
Frontend: Bootstrap 5, JavaScript
База данных: SQLite
Деплой: PythonAnywhere
